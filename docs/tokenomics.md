## Mecanica Tap-to-Earn (50%)

### Como Funciona

```
1. Usuario abre Mini-App en Telegram
2. Toca boton para "minar" puntos
3. Energia se recarga con tiempo
4. Puntos se convierten a $EVO en TGE
```

### Parametros

| Config | Valor |
|--------|-------|
| Puntos por tap | 1 |
| Energia maxima | 1,000 |
| Recarga | 1/segundo |
| Boost (ver anuncio) | +500 energia |

### Multiplicadores

```
Nivel 1 (0-10K taps):      1x
Nivel 2 (10K-50K):         1.5x
Nivel 3 (50K-200K):        2x
Nivel 4 (200K+):           3x
```
