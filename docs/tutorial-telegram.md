# Como Jugar $EVO - Guia Bolivia

**Todo dentro de Telegram. Sin apps extra. Sin wallet complicada.**

---

## Que es $EVO?

Un juego en Telegram donde ganas puntos tocando la pantalla.
Esos puntos se convierten en tokens $EVO (crypto real).

**Gratis para jugar. Sin inversion inicial.**

---

## Como Empezar (30 segundos)

### Paso 1: Abrir el Bot

```
1. Abrir Telegram
2. Buscar: @EVOtapBot
3. Tocar "Iniciar"
```

O usar link directo: t.me/EVOtapBot

### Paso 2: Empezar a Minar

```
1. Aparece boton grande de EVO
2. Tocar = +1 punto
3. Seguir tocando!
```

**Eso es todo. Ya estas minando.**

## Como Funciona

### Energia

```
Tienes energia limitada (1,000 max)
Cada tap usa 1 energia
Energia se recarga sola (1/segundo)

Tip: Vuelve cada pocas horas para mas taps
```

### Niveles

```
Mas taps = Subes de nivel = Mas puntos por tap

Nivel 1: 1 punto/tap
Nivel 2: 1.5 puntos/tap
Nivel 3: 2 puntos/tap
Nivel 4: 3 puntos/tap
```

### Referidos (La Clave)

```
Invita amigos = +5,000 puntos por cada uno
Tu amigo empieza con +2,500 puntos
Ganas 10% de todo lo que tus amigos minen!
```

**Como invitar:**
1. En el bot, tocar "Invitar"
2. Copiar tu link unico
3. Enviar a amigos por WhatsApp/Telegram

## Convertir Puntos a Crypto

### Cuando?

Cuando se lance el token (fecha anunciada en el canal).

### Como?

```
1. El bot te avisa cuando sea el momento
2. Conectas wallet de Telegram (@wallet)
3. Tus puntos se convierten a $EVO
4. $EVO aparece en tu wallet
```

### Wallet de Telegram (@wallet)

```
1. Buscar @wallet en Telegram
2. Iniciar el bot
3. Crear wallet (automatico, sin seed phrase)
4. Listo! Ya tienes wallet
```

**Sin apps extra. Sin anotar palabras. Todo en Telegram.**
