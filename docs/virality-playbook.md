## Metricas de Viralidad

### K-Factor

```
K = invitaciones enviadas × tasa de conversion

Meta: K > 1 (crecimiento viral)

Ejemplo:
- Usuario promedio invita a 5 personas
- 30% se registran
- K = 5 × 0.3 = 1.5 ✓ (viral)
```

### Tracking

```
Diario:
├── Nuevos usuarios
├── Usuarios via referido
├── Referidos por usuario
└── Conversion de invitaciones

Semanal:
├── K-factor
├── Crecimiento %
├── Retencion D7
└── Usuarios activos/total
```
