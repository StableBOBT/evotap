# EVO Token TON - Config

## Modelo (Inspirado en Notcoin/DOGS)

```
PLATAFORMA:  TON (Telegram Open Network)
TIPO:        Jetton (token nativo TON)
MECANICA:    Tap-to-Earn + Referidos
COMUNIDAD:   80% tokens
COSTO:       ~$0.33 crear token (vs $240 en BSC)
```

## Estrategia Launch

### Fase 1: Mini-App Telegram
```
1. Crear bot con mecanica tap-to-earn
2. Sistema de puntos (no tokens aun)
3. Referidos: +5,000 puntos por invitado
4. Tareas: seguir canal, compartir, etc.
```

### Fase 2: Acumulacion
```
1. Usuarios acumulan puntos
2. Leaderboard visible
3. "Seasons" de 2 semanas
4. Streaks diarios (+bonus)
```

### Fase 3: Token Launch
```
1. Crear Jetton en minter.ton.org ($0.33)
2. Airdrop: puntos → tokens (ratio anunciado)
3. Listear en STON.fi o DeDust
4. Usuarios ya invertidos emocionalmente
```

## Ventajas TON para Bolivia

```
1. Telegram nativo (ya lo usan)
2. Wallet @wallet en Telegram (sin apps extra)
3. Sin seed phrase visible (custodial simple)
4. Compra con tarjeta via Telegram
5. Gas muy bajo (~$0.01)
```
